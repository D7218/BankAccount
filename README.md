This is simple example of BankApplication  APP
A simple console-based multi-account banking system developed using Java 8,
demonstrating OOP concepts like Encapsulation, Abstraction, and Collections, 
along with Java features like Optional, Streams, and clean service-layer design.


✨ Features

✔ Multiple Account Support
✔ Account Login (Account Number + PIN)
✔ Deposit Money
✔ Withdraw Money
✔ Check Balance
✔ Transaction History (per user)
✔ Logout and Login with another account
✔ Java 8 Optional + Stream API
✔ Clean and modular service design


src/
│── BankingApp.java      # Main application (UI + Menu)
│── BankService.java     # Handles account validation
│── BankAccount.java     # Account model with balance & history


After Login Option 
1. Deposit  
2. Withdraw  
3. Check Balance  
4. Transaction History  
5. Logout // does not close the app 

 Logout

Returns to the login page (does not close the app).
New user can log in immediately.
